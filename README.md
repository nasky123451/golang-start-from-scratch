# Golang start from scratch  

# Websocket Unit

1. Go to folder

```   
cd websocket  
```   
2. Open Websocket Server (8080Port)  
```  
go run websocket_server.go   
```  
or
``` 
go run websocket_server.go -monitor 
``` 
3. Open Websocket Client Test
Single Client Test 
``` 
go run websocket_client.go  
```   
Or Brute force test   
```   
go run websocket_clients.go   
```
